# InductionProgram

A responsive and interactive College Induction Website developed using HTML, CSS, and JavaScript. The website provides information about various student clubs, societies, committees, campus facilities, FAQs, contacts, and the General Secretary team.

##  Live Demo

https://your-netlify-url.netlify.app

## Features

- Responsive design for desktop and mobile
- Modern navigation bar
- Hero section
- Student Clubs & Societies
- Committees Section
- Campus Information
- General Secretary Team
- FAQ Section
- Contact Page
- Dark Mode
- Smooth scrolling and interactive UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
